# Hotjar by Contentsquare

**Requirements**

- with PHP 8.1.0 +
- Joomla! 4.x/ Joomla! 5.x +
- J2Commerce / J2Store 4.x +





**Step 1:** Go to [J2Commerce](https://www.j2commerce.com/) > Extensions> Integrations > Hotjar by Contentsquare > click View Details > Add to cart > Checkout.

![](/img/hotjar-purchase.webp)

**Step 3:** Go to your My Downloads under your profile button at the top right corner and search for the app. Click Available Versions > View Files > Download Now

![](/img/hotjar-download.webp)
